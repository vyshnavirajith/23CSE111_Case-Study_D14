Problem Description-The Blood Bank Management System is developed to manage donor details, blood stock, donations, and blood requests efficiently. It helps in maintaining proper records and ensures availability of blood units when required.
How to run the code-
      1. Open the project in any Java IDE (IntelliJ / Eclipse)
      2. Place all files inside the same package (BloodBankManagement)
      3. Compile all Java files
      4. Run Main.java
      5. Enter input as prompted
Sample input or output     
Input:
Donor ID: 101  
Name: Rahul  
Blood Group: O+  

Output:
Donation Recorded Successfully  
Blood request created
